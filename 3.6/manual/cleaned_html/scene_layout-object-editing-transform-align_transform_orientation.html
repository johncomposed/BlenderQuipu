Align to Transform Orientation

Reference

Mode:

Object Mode and Edit Mode

Menu:

Object ‣ Transform ‣ Align to Transform Orientation

Aligns (rotates) the selected objects so that their local orientation matches the active transform orientation in the Transform orientation panel or the Orientation selection in the Transform Adjust Last Operation panels.
