Wavefront OBJ

Reference

Category:

Import-Export

Menu:

File ‣ Import ‣ Wavefront (.obj) (legacy)

OBJ format is a popular plain text format, however, it has only basic geometry and material support.

Import and export of OBJ files is built into Blender 3.3 and later without the need of an add-on. It’s documentation can be found here: OBJ Exporter.
