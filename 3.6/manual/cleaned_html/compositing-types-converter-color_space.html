Color Space Node

The Color Space Node converts images from one color spaces.

Note

Images are already automatically converted into linear color space unless specified in the imageâ€™s Color Space option.

Inputs

Image

Standard color input.

Properties

From, To

The color space of the input image and the color space to convert the image to.

The list of color spaces depends on the active OCIO config. The default supported color spaces are described in detail here: Default OpenColorIO Configuration

Outputs

Image

Standard color output.
