Instance Scale Node

The Instance Scale outputs the size of top-level instances on each axis in the local space of the modifier object.

The Instances page contains more information about geometry instances.

Inputs

This node has no inputs.

Properties

This node has no properties.

Outputs

Scale

Vector that indicates the scale of each top-level instance.
