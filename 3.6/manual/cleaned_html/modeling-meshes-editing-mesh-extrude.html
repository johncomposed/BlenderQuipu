Extrude Repeat

Reference

Mode:

Edit Mode

Menu:

Mesh ‣ Extrude ‣ Extrude Repeat

This tool behaves similar to the Array Modifier, by extruding the selection along the Z axis of the view. If the selection is not Manifold it’s extruded the specified number of times.

Offset X, Y, Z

Distance between the instances.

Steps

Number of instances.

Scale Offset

Multiplication factor to increase or decrease the offset.

See Spin.
