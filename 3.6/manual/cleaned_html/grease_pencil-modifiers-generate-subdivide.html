Subdivide Modifier

The Subdivide modifier subdivide the strokes by inserting points between other points to the lines.

Options

Subdivision Type
Catmull-Clark

The default option, subdivides and smooths the surfaces.

Simple

Only subdivides the surfaces, without any smoothing.

Subdivisions

Recursively adds more points.
