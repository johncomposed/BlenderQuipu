Points Node

The Points node generate a point cloud with positions and radii defined by fields.

Inputs

Count

The number of points to create.

Position

The position of each generated point.

Radius

The radius of each point.

Note

Since the point cloud is created from scratch, the Position and Radius inputs can only depend on the index node. Regular input nodes like the position wonâ€™t work.

Properties

This node has no properties.

Outputs

Points

Standard geometry output.
