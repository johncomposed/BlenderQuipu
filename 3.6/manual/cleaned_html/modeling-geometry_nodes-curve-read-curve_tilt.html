Curve Tilt Node

The Curve Tilt node outputs the angle used to turn the curve normal around the direction of the curve tangent in its evaluated points. Keep in mind that the output is per control point, just like the values that can be controlled in curve Edit Mode. For NURBS and BÃ©zier splines, the values will be interpolated to the final evaluated points.

The input node for this data is the Set Curve Tilt node.

Inputs

This node has no inputs.

Properties

This node has no properties.

Outputs

Tilt

The tilt angle for the normal in radians.
