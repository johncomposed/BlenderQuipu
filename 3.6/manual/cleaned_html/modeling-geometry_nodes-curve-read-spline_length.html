Spline Length Node

The Spline Length node outputs the total length of each spline, as a distance, or a number of points. This is different than the Curve Length node, which adds up the total length for all of the curveâ€™s splines.

The output values correspond to the spline domain, but the node can be used to output a value for every curve control point as well.

Inputs

This node has no inputs.

Properties

This node has no properties.

Outputs

Length

The length of each spline.

Point Count

The number of control points in each spline.
