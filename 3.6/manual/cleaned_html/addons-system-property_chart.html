Property Chart

When selecting an object you will see a new panel in the 3D Viewport’s Sidebar. In that panel you can edit the properties of selected objects.

Activation

  • Open Blender and go to Preferences then the Add-ons tab.

  • Click System then Property Chart to enable the script.
