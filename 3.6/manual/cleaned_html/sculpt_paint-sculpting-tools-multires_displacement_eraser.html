Multires Displacement Eraser

Reference

Mode:

Sculpt Mode

Tool:

Toolbar â€£ Multires Displacement Erase

This brush deletes displacement information of the Multires Modifier, resetting the mesh to a regular subdivision surface result.

This can be used to reset parts of the sculpt or to fix reprojection artifacts after applying a Shrinkwrap Modifier.

Tip

This brush works best after using Apply Base.
