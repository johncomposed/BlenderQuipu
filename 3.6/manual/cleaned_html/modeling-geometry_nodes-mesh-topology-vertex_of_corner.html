Vertex of Corner Node

The Vertex of Corner node outputs the index of the vertex that a face corner is attached to.

Inputs

Corner Index

The index of the face corner.

Note

By default this uses the index from the field context, which makes it important that the node is evaluated on the face corner domain.

Properties

This node has no properties.

Outputs

Vertex Index

The index of the vertex that the face corner is attached to.
