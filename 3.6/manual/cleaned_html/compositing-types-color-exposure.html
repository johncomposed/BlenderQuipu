Exposure Node

The Exposure Node adjusts the perceived brightness of an image by a scalar value. This node allows you to make areas of an image brighter or dimmer.

See also

The exposure can also be adjusted in the scene Color Management.

Inputs

Image

Standard color input.

Exposure

Scalar factor to adjust the exposure of the image.

Properties

This node has no properties.

Outputs

Image

Standard color output.
